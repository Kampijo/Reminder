# Reminder
![Reminder](http://i.imgur.com/vZKZp65.png) <br/>
A simple Android material design reminder app.

# Features
Allows you to add notes, remove notes, add alerts and remove alerts. <br/>
Repeated alert intervals. <br/>
Notification alerts.
